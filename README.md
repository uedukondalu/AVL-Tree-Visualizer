# AVL Tree Visualizer

This project is a visualization of a Self-Balancing Binary Search Tree (AVL Tree).

## Live Demo
https://avl-tree-visualizer.netlify.app/

## Features
- AVL Tree insertion
- Automatic balancing using rotations
- Tree visualization with edges
- Web based interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Deployment
The project is deployed using Netlify.
